* Free software: MIT license
* Documentation: https://bttt99.readthedocs.io.


Install
-------
.. code:: python

    > pip install bttt99
    # or
    > pip install -e git+https://github.com/babykick/bttt99.git@master#egg=bttt99

Usage
-----
.. code:: python

    > bttt99 rounders
